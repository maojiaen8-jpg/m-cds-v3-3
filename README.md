# M-CDS v3.3 Swimming Dashboard

Professional swimming coaching dashboard built with Next.js 14, Tailwind CSS, and React.

## Requirements

- Node.js 18+ (recommended: 20+)
- npm 9+

## Run locally

```bash
cd /Users/jianghuchuanmaomao/Documents/m-cds-v3-3
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - start production server
- `npm run lint` - run lint checks
